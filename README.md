# Task 7 - React TypeScript Practice

## How to run app
#### npm i
> installs the dependencies to the local node_modules folder

#### npm run dev
> starts local devServer in development mode

#### npm run build 
> create build in production mode

## Login credentials

this app uses DummyJson REST API data for authentication (https://dummyjson.com/users)
